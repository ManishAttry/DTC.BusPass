# DTC Bus Pass - Mock/Clone

A static HTML/CSS clone of the **Delhi Transport Corporation (DTC) e-Bus Pass** portal. This is a demonstration/mock version of the DTC bus pass system, created for educational and UI reference purposes.

## 📋 Overview

This project is a frontend mockup of the DTC Bus Pass system, showcasing a student bus pass interface. It displays a sample bus pass with details like:
- Passenger name and photo
- Pass type (Student - All route AC/Non AC)
- Bus Pass Number
- Validity period
- QR code for verification

## 📖 How It Works

The repository contains a static HTML page that:
1. Displays a DTC bus pass in a card format
2. Shows passenger details (name, pass number, validity)
3. Includes a QR code for verification
4. Uses jQuery for table pagination and interaction
5. Implements visual effects (blinking pass number, animations)

## 🚀 Usage

1. Clone or download the repository
2. Open `index.html` in a web browser
3. View the mock DTC bus pass interface

Or visit the live version: https://manishattry.github.io/DTC.BusPass/
